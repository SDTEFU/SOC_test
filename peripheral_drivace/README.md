# SOC外设库(基于SSE总线)

> ## 外设
1. [FPIOA](./FPIOA/)（现场可编程IO阵列）
2. [GPIO](./GPIO/)（通用输入输出接口）
3. [WDT](./WDT/)（看门狗）
4. [TIM](./TIM/)（定时器）
5. RCC（系统时钟）
6. UART（通用异步收发传输器）
7. [SSK](./SSK/) (同步伸缩保持串型总线)
